About GridSquid

Specifications

GridSquid is a single-page application coded in Python (Django) and JavaScript that lets a user make custom grids of images with associated sounds and captions. It was completed in 2022 as the Capstone project for Harvard's CS50 Web Programming course.

Inspiration

The idea came from a desire to teach toddlers the faces, names, and voices of family members. However, GridSquid can be used for various educational purposes, such as multi-media flashcards for homeschool curriculum, second-language studies, and other subjects for learners of all ages.

Main features include:

  • An edit mode that allows:
    • Grids to be named, expanded, resized, and sorted,
    • Tiles to be added or removed from the grid,
    • Specific tiles to have:
      • User-uploaded images (with cropping functionality),
      • User-uploaded audio files (or direct recordings using the device microphone),
      • Text captions, limited by characters, that can be previewed and edited.
  • A view mode that allows:
    • Each tile to be expanded, showing an enlarged image, existing text caption, and playing existing audio,
    • A memory game that can be played on any grid,
    • An auto-generated quiz for any grid that can be taken with audio only, text only, or both.
  • A "MyGrids" section that allows:
    • Grids to be saved and sorted into albums,
    • Multiple grids to be moved into an album at once,
    • Albums to be renamed or removed.
  • Guest accounts that allow:
    • Any user to try GridSquid without having to register for up to 30 days,
    • Any guest accounts, along with all saved grids and media, to be converted automatically to user accounts upon registration.

Future features include:

  • Ability to share grids via permanent URL
  • Invite others via email to add photos and sounds to your grid
  • Custom confirmations
